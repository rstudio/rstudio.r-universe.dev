## [rstudio.r-universe.dev](https://rstudio.r-universe.dev)

A collection of R packages from the Posit development team (previously known as RStudio).

All packages in `packages.json` are available at https://rstudio.r-universe.dev/. Learn more about [R Universe's `packages.json` format](https://docs.r-universe.dev/publish/set-up.html#subdomain).
